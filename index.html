D&D 5e Combat Simulator

Welcome to the D&D 5e Combat Simulator project!

How to Launch the Web Application:

To start the interactive web interface, please follow these steps in your terminal:

1. Navigate to the python directory:

cd python

2. Install dependencies (if you haven't already):

pip install -r requirements.txt

3. Run the Flask application:

python app.py

Once the server is running, you can access the application through your web browser.
